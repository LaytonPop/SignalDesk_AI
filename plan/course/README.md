# 课程导航

这个子目录是“按手把手方式做项目”的主线内容。你可以把它当成一份迷你实战课程。

## 推荐学习顺序

1. `phase-01-foundation.md`
2. `phase-02-crawling-and-structuring.md`
3. `phase-03-rag-and-indexing.md`
4. `phase-04-api-and-mcp.md`
5. `phase-05-reporting-and-hardening.md`

## 怎么使用这套课程文档

每一阶段都包含：

- 本阶段目标
- 你会接触的核心概念
- 需要修改或新增的代码文件
- 推荐编码顺序
- 完成后的验收标准
- 常见坑与排查建议

## 学习建议

- 每个阶段先看一遍全文，再开始写代码
- 不要跳阶段，尤其不要在抓取层没稳定前就开始调 RAG
- 每实现一个模块就运行一次，不要全部写完再调
- 自己先尝试实现，再对照现有项目代码理解设计
