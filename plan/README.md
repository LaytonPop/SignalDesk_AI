# 项目规划与课程索引

这个目录现在分成两部分：

- `参考文档`：帮你理解项目为什么这么设计
- `课程主线`：手把手带你把应用一步步做出来

如果你的目标是“跟着做出项目”，优先按课程主线推进；如果你的目标是“理解架构和面试表达”，再回来看参考文档。

## 使用方式

建议你每次只做一个阶段，并严格按这个顺序：

1. 先读本阶段目标，知道要做什么
2. 再按文档里的“要改哪些文件”逐个实现
3. 每完成一步就自测，不要攒到最后一起查错
4. 达到阶段验收标准后，再进入下一阶段
